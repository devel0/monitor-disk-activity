# monitor disk activity

![](./doc/shot.png)

## install

```sh
git clone https://github.com/devel0/monitor-disk-activity.git
cd monitor-disk-activity
npm i
npm run build
```

## usage

```sh
node dist /dev/DISK
```

or use the script

```sh
monitor-disk /dev/DISK
```
